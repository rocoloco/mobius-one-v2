## **Design System Requirements**

### **Typography**
```css
/* Import these fonts */
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&family=Poppins:wght@400;500;600;700&display=swap');

/* Use throughout */
--font-display: 'Poppins', sans-serif;
--font-body: 'Inter', sans-serif;
```

### **Colors (Use exact hex codes)**
```css
--mobius-navy: #061A40;
--mobius-teal: #048BA8;
--mobius-sage: #C1EDCC;
--background-warm: #FAFBFC;
--text-primary: #061A40;
--text-secondary: #4A5568;
--text-light: #718096;
--border-light: #E2E8F0;
```

### **Interactive States**
```css
/* Hover effects for buttons */
button:hover {
  transform: translateY(-1px);
  box-shadow: 0 4px 12px rgba(4, 139, 168, 0.3);
}

/* Focus states for form elements */
input:focus, select:focus {
  border-color: #048BA8;
  box-shadow: 0 0 0 3px rgba(4, 139, 168, 0.1);
  outline: none;
}
```

### **Key Requirements**
- All cards stack vertically on mobile
- Touch-friendly button sizes (minimum 44px height)
- Readable text on small screens
- Form controls easy to tap

## **The Steve Jobs Test**

Ask yourself after building:
1. **"Does this feel like expanding superpowers or managing infrastructure?"** ‚Üí Should feel like superpowers
2. **"Would a busy CEO understand the value immediately?"** ‚Üí Should be obvious
3. **"Does every element serve the user's business goals?"** ‚Üí Remove anything that doesn't
4. **"Does this match the Mobius 'one continuous surface' philosophy?"** ‚Üí Should feel seamless

- Replace ALL existing settings page content with this new structure
- Focus on business outcomes, not technical features
- Make it feel like empowering your AI advisor, not configuring software
- Use the exact colors and fonts specified
